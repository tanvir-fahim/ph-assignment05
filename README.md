## QNA

    01. What is the difference between var, let, and const?
        Ans: var is old version and can be reassigned and redeclared.
             let can be reassigned but can not be redeclared.
             const can't be reassigned and can't be redeclared.

    02. What is the spread operator (...)?
        Ans: spread operator is used to spread/expand elements of an array or object. To use spread operator 3 dots (...)
             is used before of an array or object.

    03. What is the difference between map(), filter(), and forEach()?
        Ans: map() transforms each element of an array and returns a new array.
             filter() selects elements based on condition and returns a new array
             forEach() is used to run a function for every element in an array and it doesn't return anything.

    04. What is an arrow function?
        Ans: Arrow function is a way of writing function in a short way. It is the modern way of writing function that
             is included in ES6.

    05. What are template literals?
        Ans: Template literals is the modern way of writing expression. We use back-tick `` for template literals and pass 
             parameter inside of it using `${}` to handle data dynamically.
